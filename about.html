Questions & answers

Is coding hard?

Coding is not hard, all it requires from you is the zeal and patience to keep learning. Some concepts could be challenging to understand, but that only means you put more effort.

What free available resources should I use to start on web development?

I would highly recommend Freecode camp which provides you with free courses to kick start your programming journey. Log in to freecodecamp and get started. I too used Freecodecamp and still do!

Can I learn all this by myself?

Yes. But it is also good to join a community of other web developers from twitter and other platforms. You get to share what you know and learn from others as well. Iron sharpens iron.

What is frontend web development?

This is programming which focuses on the user-interactive features of the website. Done through the use of HTML, CSS, JavaScript and other frontend libraries which is also the code path shown in the homepage of this website.

What is backend development?

Backend is the server side of the development focusing on APIs, servers, databases and so much more. Lets say its what you don't see but aids in the functioning of the website.

Is it really too late for me to get into tech?

It's never too late to follow your passions. You got this!
